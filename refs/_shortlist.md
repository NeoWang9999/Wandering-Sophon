# VisualSpike shortlist

这份清单用于单独列出当前最值得优先深挖、拆解、复刻或重构的站点。

## 当前优先级最高

- 001-Quantum-Periodic-Table
  - 适合方向: 量子感粒子 / 微观对象 / 发光结构
  - 为什么优先: 很贴近“一个微观存在值得被认领”的视觉吸引力。

- 002-WebGPU-Compute-Particles
  - 适合方向: GPU粒子 / 群体运动 / 微观世界动态语言
  - 为什么优先: 是这一轮最核心的交互和动态参考。

- 003-Attractors-Particles
  - 适合方向: 吸引子结构 / 聚合 / 微观核心形成
  - 为什么优先: 最适合参考“对象从粒子中凝聚出来”的过程。

- 008-Cellular-Automata-Ref
  - 适合方向: 自组织 / 细胞感 / 生成式生命结构
  - 为什么优先: 对“微观世界像活的一样”很有帮助。

- 009-Organic-Blob-Ref
  - 适合方向: 有机微生物感 / 活体轮廓 / 柔性结构
  - 为什么优先: 适合探索“可认领对象”如果更像生命体该怎么设计。
