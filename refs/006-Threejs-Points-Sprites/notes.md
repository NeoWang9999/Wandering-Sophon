# Three.js Point Sprites

- 地址: https://threejs.org/examples/webgl_points_sprites.html
- 类型: particle reference
- 风格关键词: spark, glowing, star-like, floating

适合借鉴微观粒子在黑暗背景下的漂浮感和发光感。

## 适合借鉴什么
- 微粒发光参考
- 悬浮感
- 深色背景

## 推荐重点看哪里
- 小型发光点在暗背景中的质感和密度
