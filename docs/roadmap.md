# 「流浪智子」开发路线规划
## 核心原则：视觉效果第一位，先做视觉，再做功能
先把视觉效果做到可以拍视频传播的水平，再迭代功能，优先保证传播效果，功能可以逐步上线。
---
## 🎨 阶段1：视觉Demo阶段（1周，最高优先级）
### 核心目标
优先把视觉效果做满，做出可以直接拍视频的demo版本，不需要功能，只要视觉效果到位就行。
### 具体工作（全部是P0优先级，先做这些）
| 工作内容 | 要求 | 优先级 |
| --- | --- | --- |
| 流浪智子深空渲染效果 | 2000+智子粒子在星空中自然流动、浮动，星尘粒子、流体感、氛围感拉满，达到一眼惊艳的效果 | P0 |
| 鼠标粒子流交互 | 鼠标移动、拖拽时可以和粒子流产生实时交互，支持吸引、排斥、扰动粒子流动效果，可玩性强，用户可以用鼠标“拨弄”整个星海的粒子 | P0 |
| 极端范围缩放交互 | 支持超大缩放区间：缩小到最小是全局粒子流效果，整个深空变成流动的粒子星海，可以直接玩；放大到最大可以清晰看到单个智子的量子纹路细节，方便选择点击 | P0 |
| 单个智子视觉效果 | 金属镜面球体，倒映深邃的宇宙、表面流动的量子纹路、边缘渐变光晕效果，hover时亮度提升，足够精致，简约，高级，参考 concept-images 目录下的参考图 | P0 |
| 单个智子点击交互 | 放大到单个智子后点击，自动聚焦该智子，播放二维展开动画 | P0 |
| 认领点亮动画效果 | 认领时的智子点亮、周围星尘向智子汇聚、向外扩散的光波动画，仪式感拉满 | P0 |
### 交付物
- 可运行的视觉demo，不需要后端，不需要真实认领功能，只要点击智子能播放展开、点亮动画就行
- 视觉效果达到可以直接录视频做传播素材的水平，不需要功能完整
### 验收标准
- 视觉足够浪漫、科幻感拉满，符合流浪智子的世界观
- 动画流畅，60fps运行
- 可以直接用这个demo剪宣传视频
---
## 🚀 阶段2：MVP功能阶段（第2周）
### 核心目标
视觉确定后，再做最核心的基础功能，只要能跑通完整流程就行，不需要多余功能。
### 具体功能
| 功能 | 优先级 | 说明 |
| --- | --- | --- |
| 后端数据对接 | P0 | 对接Supabase，存储智子数据、认领状态、蚀刻内容 |
| 核心认领流程 | P0 | 点击未认领智子→输入昵称→认领成功→生成专属链接，最核心流程跑通 |
| 基础蚀刻功能 | P0 | 认领后可以编辑文字内容，保存后公开可见 |
| 专属链接访问 | P0 | 通过`/s/[编号]`可以直接定位到对应的智子，展开内容 |
| 本地记忆功能 | P0 | 浏览器记住用户认领的智子，下次进入可以编辑 |
| 极简UI | P0 | 功能按钮、输入框，够用就行，不需要复杂设计 |
### 交付物
- 可公开访问的MVP版本，用户可以正常认领→蚀刻→分享
- 核心功能无严重bug
### 上线要求
- 核心流程能跑通就行，其他非必要功能全部砍掉，先上线
- 不需要限制认领数量，支持无限生成
---
## ✨ 阶段3：V1正式版本（第3-4周）
### 核心目标
上线后再迭代完善体验，增加实用功能。
### 具体功能
| 功能 | 优先级 | 说明 |
| --- | --- | --- |
| 富文本蚀刻支持 | P1 | 支持换行、插入链接、插入图片链接，丰富内容形式 |
| 随机漂流功能 | P1 | 增加「随机漂流」按钮，随机跳转到任意已认领的智子 |
| 分享功能 | P1 | 支持一键生成分享卡片，带智子预览图 |
| 移动端适配 | P1 | 适配手机端浏览和操作 |
| SEO优化 | P1 | 智子页面支持OG标签，分享时显示预览内容 |
| 数据导出功能 | P1 | 支持所有用户一键导出所有智子数据 |
### 交付物
- 稳定运行的正式版本
- 体验优化完成
---
## 🔮 后续可选迭代方向（非必须，根据用户反馈决定）
### 完全不影响平权属性的功能
- 智子自定义外观：用户可以自己调整智子的发光颜色、粒子效果
- 标签功能：用户可以给自己的智子加标签，方便搜索
- 搜索功能：支持按昵称、标签、内容搜索智子
- 评论功能：用户可以在其他人的智子页面留言
### 永远不要加的功能
- 管理员后台/内容审核功能
- 置顶/推荐/排行榜功能
- 会员/付费/广告功能
- 任何中心化管理功能
- 任何特权功能
## 风险提示
- 第一阶段优先保证视觉效果，不要纠结功能，视觉到位了就可以先剪视频预热
- 提前做好服务器扩容准备，防止视频爆火后流量超过负载
- 提前做好数据备份，定期导出所有智子数据，防止丢失
- 永远保持平权属性和浪漫感，不要加破坏氛围的功能
