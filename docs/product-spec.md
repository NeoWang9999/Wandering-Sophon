# 「流浪智子」产品规格文档
## 核心叙事（世界观）
你现在进入的是**流浪智子深空**。
你看到的不是普通粒子特效，而是在宇宙中流浪了亿万年的无主智子，它们漫无目的地在深空中漂浮、闪烁，等待第一个认领它的人类，给它一个归处。
认领成功后，你可以对它进行**二维展开蚀刻**，在它的表面刻下属于你的信息，这颗智子就会永远停留在这片深空，成为你的专属宇宙个人主页。
## 项目定位
这不是一个中心化平台，是一个**完全平权的宇宙级个人主页网络**：
- 它既是项目发起者的个人网站
- 也是所有认领了智子的用户的个人网站
- 所有智子、所有用户完全平等，没有特权，没有管理员
## 项目目标
打造一个完全开放的公共数字星海，核心价值：
1. 视觉上足够浪漫，科幻感拉满，符合流浪智子的世界观设定
2. 互动上足够轻，用户1分钟内可以完成认领→蚀刻的完整流程
3. 价值上足够实用，每个用户都可以把自己的智子作为永久个人主页使用
4. 规则上完全平权，所有用户的智子地位完全相等，没有特殊权限

## 完整用户旅程
### 新用户首次进入
1. 打开页面，看到深邃的流浪智子深空全景，无数发光智子作为粒子在星海流动
2. 可以先玩粒子流：移动鼠标/拖拽扰动粒子流动，滚轮缩小看到整个流动的粒子星海，拨弄粒子玩耍
3. 滚轮放大可以凑近看单个智子的细节，拉近时粒子流速会变慢，拉到最近几乎静止，方便找到喜欢的智子点击
4. 点击智子弹出侧边栏显示该智子的状态：已认领的直接显示蚀刻的内容；未认领智子显示「认领这颗流浪智子」按钮
### 通过分享链接进入
1. 打开他人分享的专属链接，页面默认直接放大到最大，聚焦到对应的智子，自动展开内容面板
2. 可以查看该智子的蚀刻内容，也可以滚轮逐步拉远镜头，进入宏观视角，开始玩粒子流、漫游整个星海
### 认领流程
1. 找到喜欢的未认领智子，点击「认领」按钮
2. 输入你的昵称，点击确认（不需要注册、不需要密码、不需要邮箱）
3. 选择你的智子辉光颜色（未认领智子默认白色辉光，认领后可自定义颜色）
4. 触发点亮动画：智子突然变亮为你选择的辉光颜色，表面的量子纹路全部点亮，周围星尘粒子向它汇聚，它不再流浪，有了归处
5. 认领成功，显示你的专属智子编号和永久访问链接，自动进入编辑模式
### 二维蚀刻流程
1. 认领成功后直接进入编辑模式，智子展开为二维平面，显示编辑区域
2. 你可以在上面编辑你想留下的内容：
   - 昵称/ID
   - 一句话签名/个人介绍
   - 个人简历/经历
   - 社交主页链接、作品链接、个人网站链接
   - 任何你想公开的内容
3. 编辑完成后点击「蚀刻」，内容永久保存在这颗智子上，智子收回三维状态，继续在深空中为你守候
4. 任何人点击你的智子，都可以看到你蚀刻的内容
### 认领后操作
1. 随时访问你的专属链接：`你的域名/s/[你的智子编号]`，直接定位到你的智子，展开你蚀刻的内容
2. 随时可以再次编辑你的智子内容，重新蚀刻
3. 随时可以更换智子的辉光颜色
4. 可以一键分享你的智子链接到社交平台，别人点击直接进入你的个人主页
### 探索场景
1. 在流浪智子深空中自由漫游，点击任意智子查看其他人蚀刻的内容
2. 点击「随机漂流」按钮，随机跳转到一颗已认领的智子，认识有趣的陌生人
3. 遇到喜欢的智子，可以复制它的链接分享给其他人
## 核心差异点
和普通个人主页/社交产品的最大区别：
1. **完全平权**：没有平台方、没有管理员、没有置顶、没有推荐算法，所有智子、所有用户完全平等，完全靠内容吸引访问
2. **永久有效**：认领后智子永久属于你，内容永久保存，链接永久有效，不会被删除、不会被限流
3. **浪漫载体新颖**：以三体流浪智子为概念载体，认知成本极低，自带传播属性，用户愿意分享
4. **极简流程**：不需要注册、不需要密码，输入昵称就能认领，1分钟完成全部流程
5. **既是个人也是公共**：既是项目发起者的个人网站，也是所有用户的个人网站，所有人共享同一片星海
