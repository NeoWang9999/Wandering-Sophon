# 粒子环绕运动模式研究

粒子被吸引到中心点后，可以采用以下不同的运动模式围绕中心旋转。
按实现难度从低到高排列。

---

## 1. 球壳轨道（当前方案）

**效果**：粒子约束在球面上，各自沿随机切线方向漂移。

**公式**：弹簧力指向目标半径 R，加上每粒子随机切线力。

**视觉**：电子云 / 原子模型。均匀、对称。

**问题**：所有粒子在同一半径上，缺少层次感。

---

## 2. 多层球壳

**效果**：粒子分布在多个不同半径的壳上（如 R=10, 30, 60），像原子轨道的 1s, 2s, 3p 层。

**实现**：用 `hash(instanceIndex)` 将粒子分配到不同半径：
```
shellR = 10 + hash(i) * 50  // 每粒子不同的目标半径
```

**视觉**：比单层球壳更丰富，有深度感。

**难度**：★☆☆ 改一行即可。

---

## 3. 旋涡盘 / 银河螺旋（Galaxy Spiral）

**效果**：粒子形成扁平的旋涡盘，像银河系。所有粒子在同一平面旋转，从中心向外呈螺旋臂。

**实现**：
- 将弹簧目标改为平面（Y 轴压扁）
- 切线力统一绕 Y 轴
- 加上径向螺旋偏移

```
// 压扁到 Y=0 平面
force.y += -pos.y * flattenStr
// 统一绕 Y 轴旋转
tangent = cross(vec3(0,1,0), cDir)
```

**视觉**：科幻感强。适合"星系"主题。

**参考**：[Stylized Particle Galaxy](http://www.mirzabeig.com/tutorials/stylized-particle-galaxy/)

**难度**：★★☆

---

## 4. 环面 / 甜甜圈（Torus）

**效果**：粒子形成甜甜圈形状，沿大圆和小圆两个方向旋转。

**实现**：
- 目标位置不是球面而是环面
- 大半径 R（环中心到甜甜圈中心）+ 小半径 r（管道半径）
- 每粒子以不同角度分布在环面上

```
// 环面上的目标点
theta = hash(i) * 2π          // 大圆角度
targetPos = center + R * (cos θ, 0, sin θ)
// 弹簧拉向环面管道，在管道截面内旋转
```

**视觉**：优雅，有流动感。像粒子在管道中流动。

**难度**：★★☆

---

## 5. 奇异吸引子（Strange Attractors）

**效果**：粒子不是被约束在某个形状上，而是**沿着混沌微分方程的流场运动**，自然形成复杂的 3D 轨迹。视觉极其美观，每个吸引子有独特的造型。

### 5.1 Lorenz 吸引子（蝴蝶形）
```
dx/dt = σ(y - x)
dy/dt = x(ρ - z) - y
dz/dt = xy - βz
// σ=10, ρ=28, β=8/3
```
双叶蝴蝶形状，粒子在两个叶片之间随机切换。最经典的混沌系统。

### 5.2 Thomas 吸引子（三叶草 / 柔和缠绕）
```
dx/dt = sin(y) - bx
dy/dt = sin(z) - by
dz/dt = sin(x) - bz
// b = 0.208186
```
形状像三维的柔软丝带缠绕。**非常适合艺术化粒子效果**——轨迹优雅且有对称性。

### 5.3 Aizawa 吸引子（蘑菇 / 陀螺形）
```
dx/dt = (z-b)x - dy
dy/dt = dx + (z-b)y
dz/dt = c + az - z³/3 - (x²+y²)(1+ez) + fzx³
// a=0.95, b=0.7, c=0.6, d=3.5, e=0.25, f=0.1
```
上方有一个"帽子"，下方是紧密的环。非常有立体感。

### 5.4 Halvorsen 吸引子（三翼螺旋）
```
dx/dt = -ax - 4y - 4z - y²
dy/dt = -ay - 4z - 4x - z²
dz/dt = -az - 4x - 4y - x²
// a = 1.89
```
三重对称的螺旋结构，旋转感最强。

### 5.5 Rössler 吸引子（单环 + 弹出）
```
dx/dt = -(y+z)
dy/dt = x + ay
dz/dt = b + z(x-c)
// a=0.2, b=0.2, c=5.7
```
大部分时间粒子在一个平面环上旋转，偶尔被弹到 Z 轴高处再落回。

**实现方式**（对所有奇异吸引子通用）：
```
// 把粒子当前位置代入微分方程，得到速度场
vel = f(pos)  // f 是上面任意一个 ODE 右侧
pos += vel * dt
```
不需要弹簧或切线力。粒子直接在流场中运动，自然形成吸引子形状。

**视觉**：最美观，最有动态感。粒子轨迹永不重复但始终有结构。

**参考**：
- [Strange Attractors 交互演示](https://www.dynamicmath.xyz/strange-attractors/)
- [GPU Strange Attractors (OpenFuse)](https://fusefactory.github.io/openfuse/strange%20attractors/particle%20system/Strange-Attractors-GPU/)

**难度**：★★☆（公式简单，主要是调缩放和参数）

## 6. 双螺旋 / DNA（Double Helix）

**效果**：两条螺旋线缠绕上升，粒子分两组沿各自螺旋运动。

**实现**：
```
// 粒子沿螺旋线的参数方程运动
group = hash(i) > 0.5 ? 1 : -1  // 分两组
theta = time + hash(i) * 2π
target.x = R * cos(theta)
target.y = phase * pitch         // 螺旋上升
target.z = R * sin(theta) * group
```

**视觉**：生物科技感。

**难度**：★★☆

## 推荐

| 场景 | 推荐模式 |
|---|---|
| 想要最美观的动态效果 | **Thomas 吸引子** 或 **Lorenz 吸引子** |
| 想要宇宙/科幻感 | **银河螺旋** |
| 想要简单改进当前效果 | **多层球壳** |
| 想要优雅流动感 | **环面 (Torus)** |
| 想要三维对称旋转 | **Halvorsen 吸引子** |

奇异吸引子的优势：不需要手动设计弹簧+切线+阻尼的平衡关系，公式天然产生美观的混沌轨迹。实现也比当前方案更简单——只需把 ODE 右端代入作为粒子速度即可。
