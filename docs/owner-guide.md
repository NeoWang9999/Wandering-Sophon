# 「流浪智子」所有者指南
## 📘 文档说明
本文档分为两部分：
1. 第一部分：项目发起者操作指南（给你看，包含部署、运营注意事项，符合平权设定）

## 🔧 第一部分：项目发起者操作指南
### 一、你的定位
你只是这个项目的发起者，和所有用户完全平等：
- 你也只是认领了编号为1的普通流浪智子，没有任何特殊权限
- 你没有管理员权限，不能删除、修改其他用户的智子内容
- 你不拥有这个平台，这片星海属于所有认领了智子的用户
### 二、部署指南
#### 前端部署
1. 代码开发完成后，直接部署到Vercel/Cloudflare Pages/Netlify，零成本，速度快
2. 绑定自定义域名，建议用短域名，比如`sophon.xxx`之类好记的
3. 开启CDN缓存，静态资源缓存7天，降低服务器压力
#### 后端配置
1. 注册Supabase免费版，足够支撑前10万次认领
2. 建两张表：
   - `sophons`：存储智子编号、认领状态、用户昵称、蚀刻内容、创建时间
   - `users`：不需要用户表，认领只需要存昵称即可，不需要账号系统，用浏览器本地缓存记录用户认领的智子编号，方便后续编辑
3. 不需要做管理员后台，不需要做内容审核功能，所有内容完全公开自治
### 三、规则设定（不要改，保持平权）
1. 每个浏览器只能认领1颗智子，防止刷号
2. 智子认领后永久有效，不会被回收，用户可以随时编辑内容
3. 所有智子随机分布在星海中，没有任何特殊标记、没有置顶、没有推荐
4. 编号1的智子属于你，和其他智子完全一样，没有任何特殊效果
### 四、运营指南
#### 上线初始设置
1. 上线前先认领编号1的智子，蚀刻上你自己的个人信息，作为你的个人主页
2. 初始放出10000颗智子，领完再自动生成新的，不需要限制数量
3. 不需要预留稀有编号，所有编号完全随机认领，先到先得
#### 日常运营
1. 不需要做运营、不需要做审核、不需要做管理，让星海自然生长
2. 你只需要正常用你的1号智子作为个人主页就可以
3. 如果用户有反馈，可以迭代功能，但永远不要加中心化管理功能

## 🌌 第二部分：用户使用指南（可直接放到网站上）
### 什么是流浪智子深空？
这里是一片开放的数字星海，你看到的每一个发光粒子，都是一颗在宇宙中流浪了亿万年的无主智子。
认领成功后，你可以对它进行**二维展开蚀刻**，在它的表面刻下你想留下的任何信息，这颗智子就会永远停留在这片星海，为你守候，成为你的永久宇宙个人主页，所有人都可以访问。
这是一个完全平权的公共空间，没有管理员、没有特权、没有置顶，所有智子、所有用户完全平等。
### 怎么认领流浪智子？
1. 进入网站，你会看到无数智子在星海中漂浮
2. 拖拽鼠标可以漫游整个深空，滚轮可以缩放视野
3. 找到你喜欢的未认领智子，点击它，侧边会弹出「认领这颗流浪智子」按钮
4. 输入你的昵称，点击确认，就完成认领了，不需要注册、不需要密码
### 认领后你可以做什么？
✅ 获得全球唯一的智子编号，这颗智子永远在这片星海中为你停留
✅ 获得永久宇宙个人主页链接：`你的域名/s/[你的智子编号]`，所有人都可以通过这个链接访问你的智子
✅ 对你的智子进行二维展开蚀刻，可以在上面刻下任何你想公开的内容：
   - 昵称/ID
   - 一句话签名/个人介绍
   - 个人简历/经历
   - 社交主页链接、作品链接、个人网站链接
   - 任何你想留给宇宙、留给其他人的内容
✅ 随时可以回来重新编辑蚀刻你的智子内容
✅ 自由漫游整个星海，点击其他智子查看别人蚀刻的内容
✅ 一键分享你的智子链接到社交平台
### 规则说明
1. 每个用户只能认领1颗智子，认领后永久属于你，不会被回收
2. 所有智子完全平等，随机分布在星海中，没有特殊标记、没有置顶、没有推荐
3. 所有蚀刻的内容完全公开，任何人都可以查看
4. 平台不审核内容，所有内容由发布者自行负责
5. 本项目为非盈利科幻爱好者项目，和三体官方无关
